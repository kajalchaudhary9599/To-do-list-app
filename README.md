# To-Do List App

A clean, responsive To-Do List app built with HTML, CSS, and JavaScript.

## 🔗 Live Demo
Deploy on GitHub Pages to get your live link!

## 🚀 Features
- Add tasks by typing and pressing Enter or clicking Add
- Mark tasks as complete / incomplete
- Delete individual tasks
- Filter tasks: All / Pending / Completed
- Clear all completed tasks at once
- Data saved with localStorage (persists on refresh)
- Fully responsive on mobile

## 🛠️ Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## 📁 File Structure
```
todo-app/
├── index.html   → App structure
├── style.css    → All styling
├── app.js       → All logic
└── README.md    → Project info
```

## 🏃 How to Run
1. Download or clone this repository
2. Open `index.html` in any browser
3. No dependencies or installation needed!

## 📚 Concepts Used
- DOM Manipulation
- Event Listeners
- Array methods (map, filter)
- localStorage API
- CSS animations
